Сервера
Блог сервера майнкрафт
Сервер созданный Ratwe в конце 2018 года, использующий ядро PocketMine. Основной идеей сервера было ванильное выживание без сторонних плагинов. При этом игроки должны были писать посты (рассказы) о своих приключениях для группы проекта.
UnrealSurvival
Сервер открыт Ratwe в апреле 2019 года, идея сохранилась с прошлого проекта, но на сей раз использовалось ванильное ядро на версии 1.11 (в последствии обновлен до 1.12)
RealmBlog
Сервер основанный Ratwee совместно с TheDoctorDog19 после закрытия UnrealSurvival. Основнывными игроками были люди сдружившиеся на UnrealSurvival, а после его закрытия не знающие куда пойти.
Bedrock RP
Сервер открытый TheDoctorDog19 в следствии закрытия RealmBlog. Просуществовал дольше всего (1.5 года), собрал необычное комьюнити и подарил много приключений о которых можно вспоминать с улыбкой. Первое время существования использовал ванильное ядро Minecraft Bedrock 1.12, позднее был переведен на 1.14, 1.16. Последние месяцы существования BedrockRP был перенесен на Java Edition, с использованием GeyserMC для входа с бедрока.
Vanilla Pie
Сервер основанный KonstantinFlyn после конфликта с TheDoctorDog19 и бунта, в следствии которого часть человек ушло с Bedrock RP. Использовал ванильное ядро Minecraft Bedrock 1.14 и хостинг MCPEHOST. Через некоторое время закрылся из-за отсутствия активности
Polar Bear
Сервер открытый ParussMan и kvazyyy в следствии бунта, после которого часть аудитории Bedrock RP отделилась от проекта. Работал на Minecraft Java edition и использовал GeyserMC для поддержки игры с бедрока. Через некоторое время распался из за плохого отношения kvazyyy к игрокам.
HoneyComb
Сервер открытый qIMIXERIp. Проект использовал Element Zero 1.16.20 - модифицированное ядро Minecraft Bedrock, позволяющее ограниченный моддинг. HoneyComb не был полностью ванильным сервером, на нем были введены некоторые дополнения, такие как Chisel.
BedrockRP - Новая глава
Сервер открытый TheDoctorDog19 после возвращения с желанием возродить Bedrock RP. Первое время использовал ядро Element Zero 1.16.20, после чего был перенесен на Minecraft Java Edition с поддержкой входа через GeyserMC с бедрока.
The Void
Сервер открытый qIMIXERIp в момент, когда BedrockRP - Новая глава был на грани закрытия. В последствии продолжил существовать и когда закрытие было отменено. The Void пока что находится в разработке и представляет из себя сервер на версии Minecraft Bedrock с уникальными модами (аддонами) и лором.
